## 🔄 v0.2 — 게임성 핵심 `진행 중`
> 목표: "한 번 더 하고 싶다"는 느낌. 지인 베타 테스트 가능한 수준

### Phase A — 대화 시스템 개선
- `0.2-A1` **AI 선택지 카드** — 매 턴 선택지 3개 자동 생성 + 자유입력 유지
  - 선택지는 "적극적 / 중립 / 소극적" 톤으로 3가지 제시
  - 빠른 플레이어는 카드, 몰입형 플레이어는 직접 입력
- `0.2-A2` **스테이지 고유 이벤트** — 각 스테이지마다 랜덤 돌발 상황 1회
  - 예: 카페에서 지인을 만남 / 핸드폰이 울림 / 갑자기 비가 옴
  - 이벤트마다 선택지 2개 → 호감도 ±10 분기
- `0.2-A3` **턴 압박 시스템** — 남은 턴 줄어들수록 UI 긴장감 연출

### Phase B — 엔딩 & 스토리 강화
- `0.2-B1` **숨겨진 엔딩 4종** 구현
  - 🏆 완벽 클리어: 전 스테이지 great 엔딩
  - 👫 친구로 남기: 스테이지 3 이전에 friend 엔딩 3번
  - ⚡ 속전속결: 10턴 이내 클리어
  - 💀 최악의 소개팅: 스테이지 1에서 fail 엔딩
- `0.2-B2` **준모 이력서** — 지금까지 만난 상대, 배운 교훈 누적 저장
- `0.2-B3` **엔딩 갤러리** — 달성한 엔딩 모아보기

### Phase C — 비주얼 & UX
- `0.2-C1` **캐릭터 이미지** — 유정/지은/수아 전용 이미지 (AI 생성 or 일러스트)
- `0.2-C2` **모바일 반응형** — 아이폰/갤럭시 최적화 (현재 PC 위주)
- `0.2-C3` **스테이지 전환 연출** — 페이드인/슬라이드 트랜지션
