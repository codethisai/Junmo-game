## 🏗️ 서버 아키텍처

```
현재 (v0.0~v0.2)
사용자 브라우저 → Vercel (파일 서빙) → Groq API (AI)
                                       → 브라우저 스토리지 (세이브)

목표 (v0.3~)
사용자 브라우저 → Vercel 프론트엔드
                → Vercel API Route (API 키 보호)
                  → Groq / 맥미니 EXAONE (AI)
                → Supabase DB (세이브 · 유저 데이터)

v1.0
사용자 브라우저 → Vercel 프론트엔드
                → Vercel API Route
                  → 맥미니 Ollama EXAONE 3.5-7.8B (AI · 비용 0원)
                → Supabase DB
```
