## 4. 현재 막혀있는 문제

**게임 오류:** `Failed to fetch` — AI 대화 불가

**원인:** 브라우저에서 Anthropic API 직접 호출 차단됨

**해결방법:** Google Cloud 서버에 API 중계소 설치
```
브라우저 → Google Cloud 서버(34.24.147.240) → Anthropic API
```

## 5. 지금 당장 할 것 (여기서부터 시작)

### Step 1 — Claude Code 설치 (서버에서)
SSH 접속 후:
```bash
sudo npm install -g @anthropic-ai/claude-code
claude --version  # 설치 확인
claude  # 로그인
```

### Step 2 — API 중계소 만들기
Claude Code에게 이렇게 말하기:
```
junmo-api 폴더에 Express 서버 만들어줘.
Anthropic API 키는 환경변수로 받고,
/api/chat 엔드포인트로 요청 받아서
Anthropic API로 중계해주는 서버야.
CORS 설정도 해줘.
```

### Step 3 — PM2로 서버 자동실행 설정
```bash
sudo npm install -g pm2
pm2 start index.js --name junmo-api
pm2 startup
pm2 save
```

### Step 4 — 게임 코드 수정
App.jsx에서 API 호출 주소를 변경:
```
변경 전: https://api.anthropic.com/v1/messages
변경 후: http://34.24.147.240:3000/api/chat
```

### Step 5 — 방화벽 포트 열기
Google Cloud Console → VPC 네트워크 → 방화벽 → 포트 3000 열기

### Step 6 — 테스트
게임에서 AI 대화 시도 → `Failed to fetch` 오류 사라지면 완료!
