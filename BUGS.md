# 강준모의 소개팅 — 버그 트래커

> 모든 버그는 여기에 기록. TODO.md에 버그 중복 기록 금지.
> 상태: 🔲 Open | 🔄 In Progress | ✅ Fixed

---

## 현재 보고된 버그 없음

버그 발생 시 아래 형식으로 추가:

```
## 🔲 [버그 제목] — 발견: YYYY-MM-DD

**증상**: 어떻게 나타나는가
**재현 방법**: 단계별 재현 방법
**근본 원인**: 왜 발생하는가 (진단 후 작성)
**수정**: 어떻게 고쳤는가
**PR**: #번호
**상태**: 🔲 Open → ✅ Fixed
```
